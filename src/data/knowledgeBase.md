# AML Investigations Knowledge Base

## Overview
This knowledge base contains standard operating procedures and reference materials for Anti-Money Laundering (AML) investigations at Meridian Bank.

## Investigation Framework
- **Level 1 Review**: Automated transaction monitoring alerts
- **Level 2 Review**: Enhanced due diligence and pattern analysis  
- **Level 3 Review**: Full investigation with SAR determination

## Key Regulations
- Bank Secrecy Act (BSA)
- USA PATRIOT Act
- FinCEN SAR Filing Requirements
- OFAC Sanctions Compliance
